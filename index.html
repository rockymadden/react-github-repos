Using

In addition to these quick usage examples, also check out the example directory which includes full HTML, JavaScript, and CSS.

Basic:

var GHRepos = React.createFactory(GitHubRepos);

React.render(
  <GHRepos username="username">,
  document.getElementById('ghrepos')
);

Filter (i.e. restricting repos shown):

var GHRepos = React.createFactory(GitHubRepos);
// Only show repos with a description that are not a fork.
var filter = function(repo) {
  return repo.description && !repo.fork;
};

React.render(
  <GHRepos filter={filter} username="username">,
  document.getElementById('ghrepos')
);

Check out the GitHub API documentation for the repo schema.

Map (i.e. customizing how repos are shown):

var GHRepos = React.createFactory(GitHubRepos);
// Add second paragraph with stargazer count information.
var map = function(repo) { return (
  <li key={repo.id}>
    <h2><a href={repo.homepage || repo.html_url}>{repo.name}</a></h2>
    <p>{repos.description}</p>
    <p>Stars: {repo.stargazers_count}</p>
  </li>
); };

React.render(
  <GHRepos map={map} username="username">,
  document.getElementById('ghrepos')
);
